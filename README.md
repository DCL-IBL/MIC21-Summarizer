# MIC21-Summarizer
MIC21-Summarizer is a multimodal large language model (LLM) project fine-tuned for automatic generation of image descriptions using the MIC21 dataset.​

# Features
- Multimodal LLMs trained to generate rich image descriptions from diverse inputs.​
- Utilizes the MIC21 dataset for training and evaluation.​
- Designed for academic research in computer vision, NLP, and multimodal AI.​

# Getting Started
- Prerequisites
- Python 3.8+
- PyTorch (CUDA/ROCm support), Transformers, Accelerate
- Jupyter Notebook
  
# Installation
```
git clone https://github.com/DCL-IBL/MIC21-Summarizer.git
cd MIC21-Summarizer
pip install -r requirements.txt
```

# Usage
- Launch Jupyter Notebook:
```
jupyter notebook
```
Open and run the provided notebook files to experiment with model inference and training on the MIC21 dataset.

- Dataset
A variant of [MIC21 multimodal dataset](https://huggingface.co/datasets/jkralev/mic21) should be accessed  - requires ~20GB of storage
